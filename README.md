# 🎓 Student CGPA Calculator

## 📌 Project Overview

Student CGPA Calculator is a web-based application developed using HTML, CSS, and JavaScript. It helps students calculate their CGPA by entering subject grades and credits. The application automatically computes the CGPA and updates it whenever subjects are added or removed.

## 🚀 Features

* Add subject details
* Select grades from a dropdown menu
* Enter subject credits
* Automatic CGPA calculation
* Delete subjects and recalculate CGPA
* Responsive design for desktop and mobile devices
* User-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📖 How to Use

1. Enter the subject name.
2. Select the grade obtained.
3. Enter the number of credits.
4. Click **Add Subject**.
5. Repeat for all subjects.
6. View the calculated CGPA instantly.
7. Use the **Delete** button to remove any subject and update the CGPA.
